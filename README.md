

# Medical Image Classification using CNN (InceptionV3)

## Objective
To classify medical images using deep learning and analyze optimizer performance.

## Model Used
- InceptionV3 (Transfer Learning)

## Optimizers Compared
- RMSprop
- Adam
- SGD, Adagrad, Nadam (optional experiments)

## Results
- RMSprop Accuracy: 98.51%
- Adam Accuracy: 98.01%
- F1 Score: 0.99

## Key Learnings
- Transfer learning improves convergence
- RMSprop performed best in this dataset
- Hyperparameter tuning significantly impacts performance
